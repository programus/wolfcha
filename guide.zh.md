## 1. 游戏简介

狼人杀是一款信息不对称的推理游戏。玩家分为**好人阵营**和**坏人阵营（狼人）**，通过白天互相讨论和投票、夜晚秘密行动，轮流推理、欺骗、说服对方，最终让己方获得胜利。

- **狼人**知道谁是同伴，但好人不知道谁是狼人。
- **好人**中的部分角色（神职）拥有特殊能力，可以帮助好人阵营找出狼人。
- **发言**是最重要的信息来源——每一句话都可能暴露立场或传递线索。

---

## 2. 阵营与胜利条件

| 阵营 | 获胜条件 |
|------|----------|
| **好人阵营** | 将所有狼人（包括白狼王）全部放逐出局 |
| **坏人阵营（狼人）** | 当场上存活的狼人数量 ≥ 好人数量时，狼人胜利 |

## 4. 角色说明

### 🐺 狼人（坏人阵营）

- 知道所有狼队友的身份。
- 每晚与狼队友共同商议，选定一名玩家击杀。
- 白天需伪装成好人，引导讨论方向，让好人阵营投票放逐好人。

---

### 👑 白狼王（坏人阵营）

- 与狼人完全相同，也需要伪装、参与夜晚出刀。
- **额外技能（全局仅一次）：白天自爆**
  - 在白天发言阶段主动宣布自爆。
  - 选择带走场上任意一名存活玩家（对方**没有遗言**机会）。
  - 自爆后当天**跳过投票**，直接进入下一个夜晚。
- 预言家查验白狼王的结果依然是**狼人**。

> **与普通狼人的区别**：普通狼人只能被动等待夜晚出刀；白狼王可以在白天主动出击，消灭关键目标并扰乱局势。

---

### 🔮 预言家（好人阵营）

- 每晚可以秘密查验一名玩家，得知对方是**好人**还是**狼人**。
- 查验结果只有自己知道。
- 通常需要选择合适时机公开身份，带领好人归票。
- 是狼人最优先击杀的目标之一。

---

### 🧪 女巫（好人阵营）

- 全局只有**一瓶解药**（救人）和**一瓶毒药**（毒人），各用一次、用完即止。
- 每晚最多使用一瓶药。
- **解药**：夜晚行动时可以救下被狼人击杀的玩家（包括自救）。
- **毒药**：夜晚行动时可以毒杀任意一名存活玩家（包括好人，慎用）。
- 女巫每晚可以看到当晚被狼人击杀的目标。
- 当前持有哪瓶药是**私有信息**，其他玩家只能通过局势推理判断。

> **注意**：若女巫和守卫在同一晚对同一人使用了解药和守护，该玩家**仍然死亡**（称为"毒奶"）。

---

### 🔫 猎人（好人阵营）

- 被放逐或被狼人杀死时，可以立即开枪带走场上任意一名存活玩家（**全局仅一次**）。
- **例外**：若死于女巫的毒药，则**无法开枪**。
- 开枪是猎人死亡后的强力反制手段，使用前需确认目标。

---

### 🛡️ 守卫（好人阵营）

- 每晚可以选择一名玩家进行守护，该玩家当晚不会被狼人杀害。
- **限制**：不能连续两晚守护同一名玩家。
- 可以守护自己。
- 若与女巫同时救同一人（毒奶），该玩家**仍会死亡**（见上方女巫说明）。

---

### 🃏 白痴（好人阵营）

- 第一次被投票处决时：**翻牌免疫**，不会出局，继续留在场上。
- 翻牌后永久失去投票权，但仍可正常发言。
- **例外**：被狼人击杀或被女巫毒杀时，白痴正常死亡，翻牌技能无效。

> **特点**：白痴可以在白天大胆发言，不必担心被误投出局，是好人阵营的发言担当。

---

### 🧑‍🌾 村民（好人阵营）

- 无任何特殊技能。
- 完全依靠白天的发言分析、逻辑推理和投票来帮助好人阵营获胜。
- 村民是好人人数的基础，举足轻重——一个优秀的村民善于独立判断，不轻易被带节奏。

---

## 5. 游戏流程

每一轮游戏由一个**夜晚**和一个**白天**组成，循环进行直到一方获胜。

### 第一轮特殊顺序

```
夜晚行动 → 警长竞选 → 今日死亡公布 → 白天讨论 → 投票
```

### 第二轮起的标准顺序

```
夜晚行动 → 今日死亡公布 → 白天讨论 → 投票
```

---

### 🌙 夜晚阶段

所有玩家闭眼，各角色依次秘密行动（顺序如下）：

1. **守卫**：选择一名玩家进行守护。
2. **狼人**：商议并选定击杀目标。
3. **女巫**：得知被狼人击杀的玩家，决定是否使用解药救人，或使用毒药毒杀他人。
4. **预言家**：选择一名玩家查验身份。

> 各角色的行动对其他玩家均不可见；只有系统公告（如死亡）是公开的。
>
> **特别说明**：夜晚行动按顺序依次进行，所有死亡均在**天亮时**才正式宣布。因此，当晚被狼人选为刀口的神职，**当夜仍可正常发动能力**。例如：被刀的预言家当晚仍可查验一名玩家；被刀的女巫可以看到自己是刀口，并选择是否使用解药自救。

---

### ☀️ 白天阶段

**死亡公布**：系统宣布夜晚死亡情况（姓名公开，死因统一说明为"夜间死亡"，不透露具体行凶者）。
- 若无人死亡，称为**平安夜**（可能原因：女巫救了人、守卫守护成功、或狼人选择空刀）。

**遗言**：死亡玩家可以发表遗言（最后一次发言机会）。

**特殊触发（死亡时）**：
- 若**猎人**死亡：立即宣布开枪，带走一名玩家（被毒死除外）。
- 若**白狼王**死亡（被投票出局）：可选择是否自爆（与此处不同，自爆是主动在发言时宣布）。

**讨论**：所有存活玩家轮流发言，分享线索和判断，每人每天只有一次发言机会。

---

## 6. 警长制度

**第一天**，在死亡公布之前，会先进行警长竞选：

1. **报名**：有意竞选警长的玩家举手报名。
2. **竞选发言**：报名者依次发表竞选演讲（未报名的玩家此轮不发言）。
3. **评选投票**：**未参选（警下）玩家**投票选出警长；参选玩家不参与投票。若票数相同，进行 PK 发言后重新投票；若最终仍平票，警徽**撕毁**（本局无警长）。

**警长的特权**：
- 投票权重为 **1.5 票**（即警长的一票等同于普通玩家的 1.5 票）。
- 死亡时可将警徽**移交**给任意一名存活玩家，或选择**撕毁警徽**。

> 警徽的移交方向本身也是信息——有经验的玩家会根据移交对象推断警长对局势的判断。

---

## 7. 投票规则

- 所有存活玩家参与投票，每人投票给一名玩家。
- **得票最多**的玩家被放逐出局。
- **平票处理**：平票的玩家进行 **PK 发言**，再次投票；若再次平票，重复一次；最多重投 3 次，若仍平票则**本轮无人出局**。
- 被放逐的玩家可以发表**遗言**，随后猎人可开枪。
- **白痴翻牌**：若被放逐者是白痴且尚未翻牌，则触发翻牌免疫，该玩家不出局，但失去投票权。

---

## 8. 特殊规则速查

| 规则 | 说明 |
|------|------|
| **毒奶** | 守卫和女巫在同一晚对同一人使用守护和解药，该玩家仍会死亡 |
| **猎人被毒** | 女巫毒死猎人时，猎人无法开枪 |
| **白痴转变** | 白痴翻牌后失去投票权，但仍可发言，且之后再被投出则正常出局 |
| **白狼王自爆** | 自爆带走的玩家没有遗言，且自爆后当天跳过投票直接进入夜晚 |
| **平安夜** | 无人夜间死亡；狼人自刀（击杀自己人）会导致有人死亡，不算平安夜 |
| **警徽撕毁** | 警长撕毁警徽或竞选平票后，本局无警长，无 1.5 票加成 |
| **发言顺序** | 从指定玩家开始，通常顺时针发言，警长最后发言（或由警长宣布方向） |

## 9. 常用术语表

### 基础动作

| 术语 | 说明 |
|------|------|
| **刀 / 出刀** | 狼人夜晚选定并击杀目标的行为；"刀口"指被选中的击杀目标 |
| **空刀** | 狼人夜晚放弃击杀，不选择任何目标（该夜无人被狼杀死） |
| **自刀** | 狼人夜晚选择击杀自己的狼队友，制造混乱以迷惑好人 |
| **查验 / 验人** | 预言家夜晚查验一名玩家身份的行为 |
| **跳** | 公开宣称自己的身份，例如"跳预言家"即主动表示自己是预言家 |
| **开枪** | 猎人死亡后立即带走一名玩家的行为 |
| **翻牌** | 白痴被投票出局时亮出身份牌、触发免疫的行为 |
| **自爆** | 白狼王主动宣布自爆并带走一名玩家的行为 |

### 身份与阵营

| 术语 | 说明 |
|------|------|
| **神职** | 拥有特殊能力的好人角色统称：预言家、女巫、猎人、守卫、白痴 |
| **悍跳 / 悍跳狼** | 狼人大胆声称自己是预言家（或其他神职）来扰乱局势；实施该战术的狼人称为"悍跳狼" |
| **真预（真警）** | 真正担任预言家（或警长）的玩家，区别于假冒者 |
| **假预** | 非预言家身份却声称自己是预言家的玩家，通常是狼人 |
| **狼坑** | 被多数玩家认定或高度怀疑为狼人的座位；"X号是狼坑"意为X号大概率是狼人 |

### 查验结果

| 术语 | 说明 |
|------|------|
| **金水** | 预言家查验后确认为好人的玩家；"给X号验金水"即X号是好人 |
| **查杀** | 预言家查验后确认为狼人的玩家；"查杀X号"即X号是狼人 |
| **银水** | 间接可信但未经预言家直接验证的好人；例如通过他人转述的金水（各地用法略有差异） |

### 发言与推理

| 术语 | 说明 |
|------|------|
| **带节奏** | 通过发言主导讨论方向，引导其他玩家跟随自己的立场和投票意向 |
| **甩锅** | 将怀疑转移到其他玩家身上，为自己或队友减压 |
| **站边** | 明确表态支持某名玩家（"站X号"）；先站边后倒戈称为"反水" |
| **潜水 / 深水** | 发言保守、低调，不主动表态，静观局势 |
| **冲锋** | 主动激进地发言指认某人是狼人，甘愿承担被集火的风险 |
| **倒钩** | 前期隐藏真实立场，在关键时刻突然反转、出其不意地指认狼人 |
| **逻辑压人** | 用缜密的逻辑分析施压，迫使对方解释自己的立场 |
| **狼视角** | 发言或分析无意中透露出只有狼人才会有的视角，是重要的露馅信号 |
| **对跳** | 两名玩家同时声称同一身份（通常是预言家），迫使其他人判断真假 |

### 投票与警徽

| 术语 | 说明 |
|------|------|
| **上警** | 在警长竞选阶段报名参选，成为候选人的行为 |
| **警上** | 参加了竞选的玩家，即警长候选人；警上玩家在评选投票阶段不参与投票 |
| **警下** | 未参加竞选的玩家；评选投票由警下玩家完成 |
| **归票** | 将多数票集中在同一目标上确保其出局；"警长归票X号"即建议大家都投X号 |
| **警徽流** | 解读警长死亡时传递警徽方向所代表的策略信息，推断其对局势的判断 |
| **平票 / PK** | 投票阶段出现最高票并列，进入PK发言后重新投票 |
| **撕警徽** | 警长死亡时销毁警徽，使剩余游戏中无人拥有警长加成 |

### 状态与结果

| 术语 | 说明 |
|------|------|
| **平安夜** | 当晚无人死亡，可能因女巫救人、守卫守护成功或狼人空刀造成 |
| **毒奶** | 守卫和女巫在同一晚对同一人同时守护和救人，导致该玩家仍然死亡的特殊情况 |
| **出局** | 玩家被投票放逐或夜晚死亡，退出游戏 |
| **遗言** | 玩家出局后的最后一次发言机会 |
| **翻盘** | 处于劣势的一方扭转局势取得胜利 |
